# lfs-tools

A collection of tools meant for aiding in the creation, ordering, or otherwise
of the files in the `*LFS*` books, primarily those made by the GLFS development
team.


## What is included?

In the `*LFS*` books, there are plenty of scripts and other utilities. These
are for rendering the final results, such as the HTML, wget-lists, etc. This
repository is ***NOT*** dedicated to hosting those files. Instead, this
repository hosts tools that help with working with book files, such as
generating package contents in XML or checking for updates. Hosting them in
each LFS book would lead to clutter, thus this repo aims to separate optional
tools from those that are necessary for book generation.
